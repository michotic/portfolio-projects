In-browser multiplayer fighting game ran on a NodeJS server.

Feel free to reuse any of this code to try making multiplayer games of your own.

Requires NodeJS to run 

How to Run:
1. Open "run.bat" to start server OR type "node server.js" in a command prompt inside the main folder
2. Enter "localhost:7777" into your web browser

If you want players on other networks to be able to join, edit the ip in public/main.js on line 5, and port forward 7777
